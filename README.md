# 🧮 Python 简单计算器

一个功能完整的命令行计算器程序，支持基本四则运算和连续计算。

- ✅ **基本运算**: 加减乘除 (+, -, *, /)
- ✅ **连续计算**: 使用上次结果继续计算
- ✅ **错误处理**: 除零保护和输入验证
- ✅ **友好界面**: 清晰的菜单和提示
- ✅ **重置功能**: 随时重置计算结果

## 🚀 使用方法

### 运行程序
```bash
python3 calculator.py
```

### 操作示例
```
# 基本计算
输入: 5 + 3
输出: ✅ 5 + 3 = 8

# 连续计算（使用前面的结果8）
输入: * 2  
输出: ✅ 8 * 2 = 16

# 重置结果

输出: ✅ 计算器已重置
```
